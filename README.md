# Random Wordlist Generator (Python)
A Python script that generates random passwords of variable length and saves them into a .lst file.
The program uses a wide set of alphanumeric characters and special symbols, ensures no duplicates, and lets the user choose both the output file name and the number of passwords to generate.
The program also displays a startup ASCII logo.
